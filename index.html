⚙️ Google Sheets Setup

Connect TeamPulse to your Google Sheet — one-time setup, takes 5 minutes.

1
Create a Google Sheet
Go to sheets.google.com → New blank sheet → name it TeamPulse Data. Copy the Sheet ID from the URL (the long string between /d/ and /edit).
2
Paste the Apps Script
Go to script.google.com → New project → delete all existing code → paste the entire Code.gs file (downloaded below) → replace YOUR_SPREADSHEET_ID_HERE with your Sheet ID → Save (Ctrl+S).
3
Run setupSheets → Deploy as Web App
A) Select setupSheets in the function dropdown → ▶ Run → grant permissions when prompted.

B) Click Deploy → New Deployment → ⚙ gear → Web App
   • Execute as: Me
   • Who has access: Anyone (not "Anyone with Google account")
   • Click Deploy → copy the Web App URL (ends in /exec)
⚠️ After any code change: Deploy → Manage Deployments → ✏️ Edit → New version → Deploy
4
Paste URL & Connect
Paste your Web App URL and click Test. If the test passes, click Connect.

⚡ To share with your whole team: Open the app's index.html file on GitHub, find the line var HARDCODED_GAS_URL = '...' and paste your URL there. Push to GitHub — everyone who opens the link will be auto-connected without any setup.
If still failing — open your Web App URL directly in the browser.
You should see: {"ok":true,"checkins":[...]}
If you see an error page → re-deploy the script (Step 3B again with "New version").
If you see an auth page → set "Who has access" to Anyone (not Google account).
