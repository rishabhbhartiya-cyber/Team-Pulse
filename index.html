⚙ Google Sheets Database Setup

Connect TeamPulse to your Google Sheet in 4 steps — takes about 5 minutes.

📋
One-Time Setup
After setup, all team data syncs automatically to Google Sheets.
1
Create a Google Sheet
Go to sheets.google.com and create a new blank spreadsheet named "TeamPulse Data". Copy the Sheet ID from the URL (the long string between /d/ and /edit).
2
Create the Apps Script Backend
Go to script.google.com → New project. Delete all existing code. Paste the entire contents of apps-script-backend.js. Replace YOUR_SPREADSHEET_ID_HERE with your Sheet ID. Save (Ctrl+S).
3
Run setupSheets, then Deploy as Web App
Step A — Run setup:
Select function setupSheets from the dropdown → Click  ►  Run → Click "Review permissions" → Choose your Google account → Click "Advanced" → "Go to (unsafe)" → Allow.

Step B — Deploy:
Click Deploy → New Deployment
Click the gear icon → Select Web App
Set Execute as: Me
Set Who has access: Anyone (not "Anyone with Google Account" — must be just "Anyone")
Click Deploy → Copy the Web App URL

⚠ If you update the script later, always click Deploy → Manage Deployments → Edit (pencil) → New version → Deploy to apply changes.
4
Paste Your Web App URL & Connect
Paste the deployed URL below and click Connect. All future submissions sync live to Google Sheets.
🔎 Common Connection Errors
Error: Script function not found — You pasted old code. Replace ALL code with the new apps-script-backend.js
Error: Authorization required — Set "Who has access" to Anyone (not "Anyone with Google Account")
Timeout / no response — You may need to re-deploy after running setupSheets. Use Deploy → Manage Deployments → Edit → New Version
Still failing? — Open the URL directly in your browser. You should see JSON data, not an error page.
