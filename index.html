⚙️ Google Sheets Setup

Connect TeamPulse to your Google Sheet - one-time setup, takes 5 minutes.

1
Create a Google Sheet
Go to sheets.google.com → New blank sheet → name it TeamPulse Data. Copy the Sheet ID from the URL.
2
Paste the Apps Script
Go to script.google.com → New project → paste Code.gs → replace YOUR_SPREADSHEET_ID_HERE with your Sheet ID → Save.
3
Run setupSheets → Deploy as Web App
A) Select setupSheets → Run → grant permissions.

B) Deploy → New Deployment → Web App → Execute as Me → Who has access Anyone → Deploy.
⚠️ After any code change: Deploy → Manage Deployments → Edit → New version → Deploy
4
Paste URL & Connect
Paste your Web App URL and click Test. If the test passes, click Connect to save.
If still failing - open your Web App URL directly in the browser.
You should see: {"ok":true,"checkins":[...]}
